# Minesweeper
### Minesweeper created using Java with JavaFX for the GUI.


## Controls
### Change Difficulty
>* Beginner - 'b'
>* Medium - 'm'
>* Hard - 'h'
### General Control
>* Reveal - left-click
>* Flag - right-click
>* Mass Reveal - middle-click
>* New Game - 'n'
